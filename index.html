    DWS Packages

    Monorepo for Dominus Web Services shared TypeScript packages. Built with Bun, published on npm under the @dws-std/ scope.

    Packages

    Package Description
    @dws-std/error Structured error hierarchy with UUID v7 tracking, HTTP status codes, and client/server separation.
    @dws-std/singleton-manager Centralized type-safe registry for managing singleton instances by name.

    Getting Started

    bun install

    Scripts

    Command Description
    bun run build Build all packages
    bun run test Run all test suites
    bun run lint Lint all packages
    bun run lint:fix Lint and auto-fix
    bun run docs Generate TypeDoc documentation
    bun run clean Remove all node_modules and bun.lock files
