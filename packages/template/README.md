# 📦 Package Template

Scaffold for new `@dws-std/` packages. Copy this directory and fill in the blanks.

## 🔧 Installation

```bash
bun add @dws-std/your-package-name
```

## ⚙️ Usage

```ts
import { YourExport } from '@dws-std/your-package-name';
```
